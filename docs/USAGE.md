## 首页与导航

### 侧拉菜单

点击首页左上角 **菜单按钮**（汉堡图标）打开侧拉菜单。

侧拉菜单按功能分为 4 个分组：

| 分组 | 菜单项 | 用途 |
|------|--------|------|
| **娱乐** | 今日秘书舰 / 查看 Live2D | 秘书舰设置、L2D 浏览 |
| **挑战** | 看图识舰娘 / 听音识舰娘 | 小游戏 |
| **工具** | 碧蓝航线助手 / 啾信 | 玩家查询、AI 对话 |
| **系统** | 设置 / 关于 | 应用配置 |

**操作说明**：
- 点击菜单项：关闭抽屉并跳转到对应页面
- 点击关闭按钮（右上角 ✕）：仅关闭抽屉
- 手势：可从屏幕左边缘向右滑动打开，从右向左滑动关闭
- 切到横屏时，抽屉自动关闭

### 主题切换

应用支持两种主题风格：

1. **经典风格**（默认）：紫色 Material Design 配色，圆角矩形
2. **指挥中心（Command Center）**：青蓝色 HUD 风格，切角矩形 + 切角面板 + 金色高亮

切换路径：侧拉菜单 → 设置 → 界面风格 → 指挥中心 UI 开关

> 提示：所有界面元素（颜色、间距、形状、动画）都会随风格自动适配。

## 船坞与舰娘管理

### 浏览舰娘

1. 打开应用默认进入 **船坞**（首页）
2. 顶部有筛选工具栏，支持：
   - **阵营筛选**：白鹰 / 皇家 / 重樱 / 铁血 / 北方联合 / 撒丁帝国 / 自由鸢尾 / 维希教廷 / 东煌 / 游离 / 海上传奇
   - **类型筛选**：驱逐 / 轻巡 / 重巡 / 战巡 / 战列 / 航母 / 轻航 / 维修 / 潜艇 / 炮巡 / 防驱
   - **稀有度筛选**：海上传奇 / 决战方案 / 超稀有 / 最高方案 / 精锐 / 稀有 / 普通
3. 点击搜索框可按舰娘名称搜索
4. 列表支持滚动加载

### 舰娘卡片交互

- **单击卡片**：进入舰娘详情页，浏览立绘和语音
- **长按卡片**：弹出快捷操作菜单
  - 收藏 / 取消收藏
  - 查看详情
  - 随机播放语音

### 收藏舰娘

将舰娘添加到 **后宅**（个人收藏夹），便于快速访问：

1. 在舰娘卡片长按 → 选择 **收藏**
2. 或在舰娘详情页点击右上角的 ⭐ 按钮
3. 收藏状态会持久化保存在 Room 数据库

### 详情页功能

- 浏览多张皮肤立绘
- 按皮肤切换语音列表
- 播放单个语音
- 加入稍后播放队列
- 收藏 / 取消收藏

## 后宅（收藏）

通过侧拉菜单 → 后宅 / 底部导航 → 后宅 进入。

- 展示所有已收藏的舰娘
- 卡片支持与船坞相同的交互
- 取消收藏的舰娘会从此页消失（但不会从船坞中删除）

## 语音播放

### 基础播放

1. 在语音列表或舰娘详情页点击 **▶ 播放** 按钮
2. 播放控制栏会出现在页面底部，包含：
   - 进度条 + 时间显示
   - 上一首 / 播放暂停 / 下一首
   - 播放模式切换
   - 稍后播放队列入口
3. 播放时支持：
   - **后台播放**：切到后台或锁屏后继续播放
   - **通知栏控制**：下拉通知栏可看到播放器
   - **锁屏控制**：锁屏界面可控制播放
   - **耳机线控**：支持播放/暂停、上一首/下一首

### 稍后播放队列

将多个语音加入队列，按顺序连续播放。

**添加条目**：
1. 在语音列表长按某条语音 → 选择 **稍后播放**
2. 或在舰娘详情页语音项的菜单中点击 **稍后播放**

**管理队列**：
1. 点击播放控制栏的 **队列按钮**（播放列表图标）
2. 队列弹窗中：
   - **正在播放**的条目带脉冲动画和主题色描边
   - **下一个**播放的条目带 "Skip Next" 图标
   - **已播放**的条目透明度降低，并带 "已播放" 徽章
   - **未播放**的条目可点击单独播放
3. 弹窗顶部按钮：
   - **播放全部**：从队首开始顺序播放
   - **清空**：清空整个队列
4. 每个条目右侧的 **× 按钮** 可移除该条目

> 数据持久化：稍后播放队列保存在 DataStore 中，应用重启后仍保留。

### 播放模式

点击播放控制栏的 **模式按钮** 可循环切换：

| 图标 | 模式 | 说明 |
|------|------|------|
| 🔁 | 列表循环 | 播放完列表后从头开始 |
| 🔂 | 单曲循环 | 一直重复当前首 |
| 🔀 | 随机播放 | 随机选择下一首 |

## 秘书舰模式

设置一个常驻的舰娘，让她在屏幕上陪伴你。

### 设置秘书舰

1. 侧拉菜单 → **今日秘书舰**
2. 选择一个舰娘作为秘书舰
3. 确认后，应用启动时会显示该舰娘的小人立绘

### 小人互动

- **拖动**：按住小人可拖到屏幕任意位置
- **拖出屏幕**：可隐藏到屏幕边缘
- **点击**：随机播放该舰娘的一条语音
- **长按**：弹出设置菜单，可更换秘书舰或关闭

### 系统悬浮窗（高级）

在设置中开启 **悬浮窗权限** 后，秘书舰可以：
- 在其他应用上方显示
- 随时点击触发语音

> 注意：需要 Android 11+ 的"显示在其他应用上方"权限。

## 小游戏

### 看图识舰娘

1. 侧拉菜单 → **看图识舰娘**
2. 系统会展示一张立绘（隐藏名称）
3. 从 4 个选项中选择正确的舰娘
4. 答对得分，答错展示正确答案

### 听音识舰娘

1. 侧拉菜单 → **听音识舰娘**
2. 系统会播放一段语音
3. 从 4 个选项中选择正确的舰娘
4. 答对得分

## 碧蓝航线助手

通过碧蓝航线官方 API 查询玩家信息。

### 配置

首次使用前需要配置：
1. 侧拉菜单 → **设置** → **碧蓝航线助手**
2. 填入：
   - **默认 UID**：游戏内玩家 UID
   - **默认服务器**：支持名称（如 `秋季旅行`）或 ID（如 `120`）

### 查询玩家

1. 侧拉菜单 → **碧蓝航线助手** → **查玩家**
2. 点击 **查询** 按钮
3. 返回信息包括：
   - 等级、经验、UID
   - 收集率、舰娘数量
   - 金币、钻石、石油、零件
   - 委托进度
   - 科研进度
   - 待办副本

### 查询建造

1. 切换到 **查建造** 标签
2. 点击 **查询建造记录**
3. 返回最近的建造记录（支持分页，最多 500 条）

## 啾信（AI 对话）

与 AI 角色扮演的舰娘进行对话。

### 配置 API

1. 侧拉菜单 → **设置** → **啾信配置**
2. 填入：
   - **API Key**：大语言模型 API 密钥（OpenAI 兼容）
   - **API URL**：API 端点
   - **模型**：模型标识（如 `gpt-4`）
3. 保存配置

### 开始对话

1. 侧拉菜单 → **啾信**
2. 选择一个舰娘人设
3. 在底部输入框中输入消息
4. AI 会以该舰娘的角色风格回复

### 对话管理

- 支持多轮上下文
- 支持重新生成回复
- 支持清空对话历史
- 支持导出对话为文本

## Live2D 浏览

浏览社区 Live2D 模型资源。

### 进入浏览

1. 侧拉菜单 → **查看 Live2D**
2. 应用会加载 [l2d.su](https://l2d.su/) 网站
3. 应用内置了对 `l2d.su` 域名的 SSL 证书信任，确保正常访问

### 交互

- 滚动浏览模型列表
- 点击模型查看详情
- 支持在 WebView 内放大查看
- 支持下载模型

### 添加自定义 L2D 站点

如需访问其他 Live2D 站点：

1. 设置 → 高级 → 网络安全
2. 添加需要信任的域名
3. 重启应用生效

> ⚠️ 仅添加你信任的网站，不安全的 HTTPS 证书会带来中间人攻击风险。

## 故障排除

### 应用无法播放语音

- 检查网络连接
- 确认 API 域名未被防火墙拦截
- 在 设置 → 高级 → 数据管理 中清除缓存

### Live2D 页面无法加载

- 确认网络可以访问 `l2d.su`
- 检查网络安全配置中的域名白名单
- 尝试在浏览器中访问 [l2d.su](https://l2d.su/) 验证

### 碧蓝航线助手查询失败

- 确认 UID 和服务器配置正确
- 确认游戏内玩家数据可公开查询
- 检查网络连接

### 啾信对话无响应

- 确认 API Key 有效且余额充足
- 确认 API URL 可访问
- 在 设置 → 高级 → 数据管理 中清除缓存

### 小屏设备上显示异常

应用已对屏幕最小宽度 ≤ 360dp 的设备做了自适应。如仍有问题：
- 在设置 → 界面风格 中切换主题
- 重启应用
